# Specula
## Description
See the private repo for the language description

## Requirements
1. C++
2. CMake
3. gcc/clang

## Installation
1. Build cmake files
```
cmake -B build
```
2. Build the project
```
cmake --build ./build
```
3. Run the project
```
./build/specula
```
## Running Tests
The tests are located in the `tests` folder and the build files in the `build/tests` folder. To run it:
```
./build/tests/specula
```
